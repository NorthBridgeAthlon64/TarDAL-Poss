本文件夹作用
----------
评委演示包中的「示例红外/可见光图像对」模板目录。可与项目根目录下的 Picture-Example 同时使用：构建时由 scripts\Copy-PictureExampleToDist.ps1 先将根目录 Picture-Example 内容合并到 dist\TarDAL-Poss-Backend\Picture-Example\，再合并本目录内容（同名文件可被后者覆盖），最终与 TarDAL-Poss-Backend.exe 位于同一文件夹，便于评委打开目录拖图到网页上传。

各文件说明
----------
readme.txt
  本文件，说明本文件夹用途及与根目录 Picture-Example 的合并关系。

（说明）除本 readme 外，本目录内其余文件由参赛队自行添加，例如各组示例图的红外图、可见光图；文件名与数量可自行约定，须符合网页上传格式要求。
